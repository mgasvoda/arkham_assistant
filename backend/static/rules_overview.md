# Arkham Horror LCG Rules Overview

This file will contain key game rules for the AI agent to reference.

## Action Economy

- Each investigator gets 3 actions per turn
- Actions can be used to: Investigate, Move, Fight, Evade, Draw, Resource, Play card, Activate ability

## Resource System

- Investigators gain 1 resource per turn
- Cards cost resources to play
- Some cards provide additional resources

## Deck Construction

- 30 cards minimum
- 0-2 copies of each card (by title, not by level)
- Must respect investigator's deckbuilding restrictions
- Include required signature cards

## Skill Tests

- Base difficulty vs. investigator's skill value
- Modified by card effects, tokens, and chaos bag
- Success = meet or exceed difficulty
